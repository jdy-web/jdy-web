## 名词解释

| **术语** | **Emoji** | **解释**          |
| ------ | --------- | --------------- |
| 必须     | ✅         | 除非是非常特殊的情况，必须执行 |
| 禁止️    | ⛔         | 除非是非常特殊的情况，禁止使用 |
| 建议     | 👍        | 建议这样使用，但没有那么严格  |
| 不建议    | 🤔        | 不建议这样使用，但没有那么严格 |
| 可选     | 🔄        | 在你的模块/领域下保持一致即可 |

参考：https://datatracker.ietf.org/doc/html/rfc2119
